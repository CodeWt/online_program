# online_program
>simple description
```aidl
1.该项目是一个Python的在线编辑运行平台,
通过在线创建工程,编辑代码,调试运行,
实现可以在网页进行编程。
2.可搜索安装运行所需要的依赖包
3.可保存自己的代码,并查看历史代码
4.也可以从平台检索API,借鉴参考他人
```
